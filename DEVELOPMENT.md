# Development

## Prerequisites

```bash
pnpm install
```

## Development

serve

```bash
pnpm serve -F picgo-plugin-app
```

dev

```bash
pnpm dev -F picgo-plugin-app
pnpm dev -F picgo-plugin-bootstrap
```

## Build

```bash
pnpm package
```

artifacts structure

```
├── build
  ├── package.zip
```
